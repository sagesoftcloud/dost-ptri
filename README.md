# DOST PTRI - AWS Fundamentals Training Program

> ⚠️ **For training purposes only.** The materials in this repository are designed for hands-on lab exercises as part of the DOST PTRI AWS Fundamentals Training Program.

## Labs

| Lab | Description |
|-----|-------------|
| [EC2 Auto Scaling & Load Balancing](./ec2-autoscaling-loadbalancing/) | Build a multi-tier architecture with VPC, EC2, ALB, and Auto Scaling |
| [Lambda Serverless](./lambda-serverless/) | Three hands-on Lambda labs: Hello World, Calculator, and S3 Image Processor |
| [API + Cognito + SQS + SNS](./api+cognito+sqs+sns/) | REST API with Cognito auth, SQS order queue, SNS fan-out, and React integration app |
| [Bedrock GenAI](./bedrock-genai/) | Building with Amazon Bedrock — foundation model APIs, chatbots, RAG, image generation, and agentic AI |
| [SageMaker Walkthrough](https://github.com/sagesoftcloud/dost-ptri-day5-sagemaker-walkthrough) | SageMaker console tour — Studio, JumpStart, Jobs, Models, Endpoints, Pipelines (no deployment, $0 cost) |
