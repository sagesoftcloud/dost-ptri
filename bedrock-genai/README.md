### Prerequisites

- AWS account with Bedrock model access enabled
- Region: **us-east-1** (N. Virginia) — widest model availability
- Python 3.9+ installed locally
- VS Code (recommended)

### ⚠️ Cost

Amazon Bedrock uses **token-based pricing** — you pay only for what you use. The workshop labs cost **less than $1 USD** in total for typical usage.
