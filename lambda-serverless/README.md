# AWS Lambda Serverless Labs

## DOST PTRI - AWS Fundamentals Training Program

Three hands-on labs using AWS Lambda with Python — all done via the AWS Console (ClickOps).

| # | Lab | Services Used | Time |
|---|-----|--------------|:----:|
| 1 | [Hello World](./01-hello-world.md) | Lambda | 10 min |
| 2 | [Calculator](./02-calculator-api.md) | Lambda | 15 min |
| 3 | [S3 Image Processor](./03-s3-image-processor.md) | Lambda, S3, IAM, CloudWatch | 25 min |

### Prerequisites

- AWS account with AdministratorAccess
- Region: **ap-southeast-1** (Singapore) or **us-east-1** (N. Virginia)

### Naming Convention

All resources: `dost-ptri-lambda-[yourname]-<resource>`

### ⚠️ Cost

Lambda free tier includes 1M requests/month and 400,000 GB-seconds. These labs will cost **$0** if you're within free tier.
